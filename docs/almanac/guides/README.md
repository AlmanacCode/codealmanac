# Guides

`guides/` explains how to perform recurring maintainer tasks. A guide should
state the goal, prerequisites, commands or files involved, expected output, and
verification.

Good guide topics include setup, debugging search, fixing a page, reviewing wiki
quality, running lifecycle jobs, shipping a release, or migrating legacy wiki
content.
