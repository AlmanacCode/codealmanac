# Decisions

`decisions/` records accepted choices that should shape future work. A decision
page should say what was chosen, what alternatives were plausible, why the
choice was made, what constraints it creates, and what evidence would reopen it.

Do not put every brainstorm here. A decision page is for settled project
knowledge.
